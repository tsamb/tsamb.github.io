javascript game paying homage to total annihilation

Tuesday 27 January, 2015

I'm building a game in the browser. It is inspired by the resource system of Chris Taylor's 1997 game Total Annihilation.

This is one side project that I have been really amped to be working on. Often I'll lose interest with toy projects after a few days. This one, I'm excited to get home and work on.

It's almost ready to be released into the wild as an alpha release. At least, right now, all of the MVP mechanics are there like resource generation, hp, damage etc. And I'm having fun playing it locally.

I'll probably throw it on this domain until I want to score games and store those scores in some backend.
